# Бот-Ассистент помощник

Я писал (и еще пишу) бота для студии-стрижки, студия регистрирует клиентов по телефону, бот создается чтоб у клиентов была возможность ознакомиться с услугами и записаться на них в любое время.

### Для клиента

Клиент через бота может посмотреть интересующиеся услуги, акции, создать свой профиль для обратной связи, записаться на услугу, просмотреть свои записи.

По необходимости, клиент может удалить свой профиль и удалением остановить бота.

### Для мастера

Мастер через бота может просмотреть базу клиентов, увидеть зарегистрированные записи, создавать и отправлять рассылку о свободном времени или акции, редактировать записи клиентов, обновлять клиентскую базу.

### Таблицы в базе:
* Клиенты (users)
* Женские категории (female_categories)
* Женские услуги (female_items)
* Мужские категории (male_categories)
* Мужские услуги (male_items)
* Детские категории (child_categories)
* Детские услуги (child_items)
