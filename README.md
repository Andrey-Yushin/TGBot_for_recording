# Бот-Ассистент помощник

Я писал (и еще пишу) бота для студии-стрижки, студия регистрирует клиентов по телефону, так же студия использует платное приложение, через которое клиент может выбрать для себя услугу.

Бот создается чтоб у клиентов была возможность ознакомиться и выбрать себе услугу в любое время.

### Для клиента

Клиент через бота может посмотреть интересующие услуги, акции, создать свой профиль для обратной связи, выбрать услугу для себя, просмотреть свои записи.

По необходимости, клиент может удалить свой профиль и удалением остановить бота.

### Для мастера

Мастер через бота может просмотреть базу клиентов, увидеть зарегистрированные записи, создавать и отправлять рассылку о свободном времени или акции, редактировать записи клиентов, обновлять клиентскую базу.

### Таблицы в базе:
* Клиенты (users)
* Категории услуг (female_categories/male_categories)
